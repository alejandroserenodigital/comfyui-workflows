## What's in this repo

* **ComfyUI workflows (JSON):** ready to load graphs for image to video, video to video, and stylized cinematic output
* **LoRA setups:** configurations for consistent, on brand looks
* **Prompt guides:** repeatable prompt structures others can replicate

## How to use

1. Open ComfyUI
2. Load any JSON file from this repo (drag it onto the canvas or use Load)
3. Swap in your own inputs and prompts following the notes in each workflow
